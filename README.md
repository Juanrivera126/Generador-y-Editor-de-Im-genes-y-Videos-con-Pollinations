# Generador y Editor de Imágenes y Videos con Pollinations

Esta aplicación es una herramienta avanzada basada en la web que utiliza la potencia de **Pollinations AI** para la generación y edición de contenido visual. Ha sido diseñada con un enfoque en la experiencia de usuario (UX) premium, ofreciendo una interfaz limpia, compacta y altamente funcional.

## Características Principales

### 🖼️ Generación y Edición de Imágenes
- **Flujo de Trabajo Robusto**: Genera una imagen inicial y edítala mediante lenguaje natural.
- **Bloqueo Inteligente**: Durante la fase de edición, los controles de configuración se inhabilitan para evitar errores de estado, permitiendo un enfoque total en los cambios.
- **Comparación Dinámica**: Visualiza la imagen original y la editada simultáneamente con una barra deslizante (slider) vertical para apreciar cada detalle del cambio.
- **Panel de Acciones Rápido**: Botones de "Abrir" y "Descargar" integrados directamente en la vista comparativa con un diseño ultra-compacto.

### 🎥 Generación de Videos
- **Múltiples Fuentes**: Crea videos a partir de un prompt de texto o utiliza la **imagen original** o la **imagen editada** como punto de partida visual.
- **Fusión Creativa (Prompt + Imagen)**: Una característica destacada de esta herramienta es que, al seleccionar una imagen (original o editada) como base para el video, el sistema la **combina dinámicamente con el texto del prompt**. Esto permite usar una imagen estática y guiar su evolución o estilo mediante instrucciones de texto, logrando resultados visuales sorprendentes y únicos.
- **Configuración Detallada**: Ajusta el modelo, dimensiones y duración del video.

### 🛠️ Tecnología y Diseño
- **Motor de IA**: Integración con Pollinations AI (API REST).
- **Asistencia**: Desarrollado con el apoyo de **Gemini 1.5 Flash** y **Antigravity**.
- **Diseño Responsive**: Interfaz optimizada para aprovechar al máximo el espacio vertical, eliminando paddings innecesarios y manteniendo los elementos clave siempre a la vista.
- **Multi-idioma**: Soporte completo para Español, Inglés y Francés.

## Cómo Usar
1. **Configura tu API Key**: Introduce tu clave de Pollinations en la pestaña de Imágenes (puedes obtener una fácilmente desde el botón integrado).
2. **Genera**: Elige un modelo, escribe un prompt y pulsa "Generar Imagen".
3. **Edita**: Una vez tengas tu imagen, escribe los cambios deseados en el cuadro de edición a la izquierda y pulsa "Aplicar cambios".
4. **Crea Video**: Si deseas darle movimiento, ve a la pestaña de Videos y selecciona si quieres usar tu imagen original o la editada como base.
